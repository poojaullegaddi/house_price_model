# house_price_model
# 🏠 House Price Prediction App using Linear Regression

## 📌 Project Overview
This project is a Machine Learning web application that predicts house prices based on various input features such as income, number of rooms, and location.

The model is built using Linear Regression and deployed as an interactive web app using Streamlit.

---

## 🚀 Features
- Predict house prices in real-time  
- Simple and user-friendly UI  
- End-to-end Machine Learning project  
- Deployed using Streamlit  

---

## 🧠 Machine Learning Workflow
1. Data Collection  
2. Data Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Model Training (Linear Regression)  
5. Model Evaluation  
6. Model Saving using Pickle  
7. Deployment using Streamlit  

---

## 📊 Dataset
- California Housing Dataset  

### Features:
- Median Income  
- House Age  
- Average Rooms  
- Average Bedrooms  
- Population  
- Average Occupancy  
- Latitude  
- Longitude  

### Target:
- House Price
